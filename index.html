ETALKING CRM TOOL

名單管理快速工具

一鍵壓紀錄、倒數噴單警示、新單進度追蹤。讓名單管理更有效率。

// 安裝書籤
📋 名單管理工具

將左側按鈕拖曳到瀏覽器書籤列

拖曳放開後即完成安裝，之後點一下就能啟動。

// 使用步驟
01

顯示書籤列

按 Ctrl+Shift+B 確認書籤列已顯示於瀏覽器上方。

02

拖曳安裝

將綠色「📋名單管理工具」按鈕直接拖曳到書籤列放開即安裝完成。

03

登入後台點擊書籤

登入 etalking 後台後，在任意頁面點一下書籤，按確定進行跳轉。

04

再點一次書籤

跳轉完成後，再點一次書籤，名單管理面板即會自動載入。

// 跳轉狀態說明
✅
正常啟動

已在正確頁面，面板直接載入，無需任何額外操作。

🔄
需要跳轉

點確定後自動跳轉，跳轉完成再點一次書籤即可。

⚠️
通道失效

請先在名單頁面點開任一張單的「編輯」，再重新點書籤。
